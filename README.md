# Google Play Store Analysis

A data analytics project that analyzes Google Play Store applications using Python to uncover insights through data cleaning, exploratory data analysis (EDA), and visualizations.

## Dataset

* Google Play Store Data.csv
* User Reviews.csv

## Tech Stack

Python • Jupyter Notebook • Pandas • NumPy • Matplotlib • Seaborn • Plotly

## Key Highlights

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Category-wise Analysis
* Rating & Review Analysis
* Install Trend Analysis
* Data Visualization

## Getting Started

1. Clone the repository.
2. Install the required dependencies.
3. Open `analysis.ipynb` in Jupyter Notebook.
4. Run all cells.
